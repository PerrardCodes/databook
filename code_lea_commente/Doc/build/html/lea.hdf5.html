lea.hdf5.h5py_convert.h5py_in_Data(f)¶

Prend en paramètre un fichier HDF5 et renvoie le fichier Data qui est dedans

Paramètres:f (Files h5py) – Fichier ouvert
Renvoie:Data Object

Avertissement

Ne marche pas récursivement ! Le sous-dossier Data doit être à la racine.

Note

Si le fichier HDF5 contient un objet Mesure il faut utiliser h5py_in_Mesure()

lea.hdf5.h5py_convert.h5py_in_Mesure(f)¶
Paramètres:f (Files h5py) – Fichier ouvert
Renvoie:Mesure Object
  • Si jamais le fichier a un sous dossier Contour il va appeler h5py_in_Contour()
  • Si jamais le fichier a un sous dossier Bulles il va appeler h5py_in_Bulles()
  • Si jamais le fichier a un sous dossier PIV3D il va appeler h5py_in_piv3d()
  • Si jamais le fichier a un sous dossier Volume il va appeler h5py_in_Volume()

lea.hdf5.h5py_convert.obj_in_h5py(object, file, group=None, point='', attr='')¶
Paramètres:
  • object (Python Object) – Objet à transférer en HDF5
  • file – fichier dans lequel il faut sauvegarder le fichier hdf5
Renvoie:

None

Ecrit l’objet dans le fichier au fur et à mesure qu’il descent récursivement dans l’objets passé en paramètre.

Necessite l’utilisation de ouverture_fichier() ou de file_name_in_dir()

Note

Les paramètres group, point, attr sont utilisés pour les récursions.

Sauvegarde des données depuis un objet¶

Créer un fichier HDF5 depuis un objet

#Je prend ici l'exemple d'un objet Data, mais cela marche aussi avec un objet Mesure ou un objet de votre création
import lea.hdf5.h5py_convert as lh5py
import lea.data.Data as ldata

adresse = "/home/ldupuy/Documents/"
d = ldata.Data(fichier, param, spec, date=date, heure=heure)

f = lh5py.file_name_in_dir(d, adresse)
lh5py.obj_in_h5py(d, f)
f.close()

Créer un fichier HDF5 depuis un sous-objet de Mesure

import lea.hdf5.h5py_convert as lh5py
import lea.mesure.Mesure as lmesure
import lea.mesure.Volume as lvolume
import lea.data.Data as ldata

adresse = "/home/ldupuy/Documents/"

d = ldata.Data(fichier, param, spec, date=date, heure, heure)
m = lmesure.Mesure(d)
v = lvolume.Volume(d)
#Faire les actions qu'on veut sur les volumes.#
m.add_measurement(v)
f = lh5py.file_name_in_dir(m, adresse)
lh5py.obj_in_h5py(m, f)
f.close()

Avertissement

Mieux vaut sauvegarder le sous-objet de Mesure dans Mesure pour le hdf5

datafile="/home/ldupuy/Documents/20181220_1_test1.hdf5"
